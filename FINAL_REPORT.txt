         ▼

         │

  Parse response.data.item.episodes[0].server_data
         │
         ▼
  setItem() and setCurrentEp() with first or URL episode
         │
         ▼
  EpisodeScroll renders with episodes array
         │
         ▼
  User selects episode → handleSelectEpisode()
         │
         ▼
  HLS.js loads link_m3u8, video plays
         │
         ▼
  Watch progress saved every 30 seconds
         │
         ▼
  On video end: auto-play next or show countdown

💾 WATCH HISTORY TRACKING

  Save Function: saveWatchProgress(movieId, episodeName, currentTime, duration)
  Get Function:  getWatchProgress(movieId, episodeName)
  
  Data Per Episode: {
    key: "{movieId}_{episodeName}",
    movieId, episodeName,
    currentTime, duration,
    percentage,
    timestamp,
    movieInfo: { title, poster, slug }
  }
  
  Features:
  • Saves every 30 seconds
  • Resume if 5-95% watched
  • Max 100 entries
  • Per-episode tracking
