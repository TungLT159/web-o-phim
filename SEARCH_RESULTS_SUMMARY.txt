### API ENDPOINTS

**Base URL**: https://ophim1.com/

| Endpoint | Purpose | Function |
|----------|---------|----------|
| /v1/api/phim/{id} | Get movie details + episodes | detail() |
| /v1/api/danh-sach/{type} | Get movie list | getMoviesList() |
| /v1/api/the-loai/{type} | Get by genre | getListByType() |

## KEY INSIGHTS

1. **Episodes are stored in an array** under esponse.data.item.episodes[0].server_data
2. **Only first server is used** (supports multiple servers for fallback)
3. **Episode identifier is string** (not numeric) - "1", "2", "3"
4. **URL state is maintained** for specific episode selection
5. **Auto-play is configurable** via localStorage
6. **Watch progress is per episode** and stored locally
7. **Fallback chain**: HLS.js → link_m3u8 → link_embed
8. **Keyboard support** for TV and desktop navigation
